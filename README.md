# Симуляция

Пошаговая симуляция 2D мира, населённого травоядными и хищниками. Построена на принципах ООП.

![demo](.github/assets/simulation.gif)

## О проекте

Мир представляет собой матрицу **N×M**, где каждая клетка занята одним объектом. Каждый ход существа ищут цель с помощью **алгоритма поиска в ширину (BFS)** и двигаются к ней. Симуляция работает бесконечно, пополняя траву и травоядных, если их становится слишком мало.

## Объекты мира

| Объект | Символ | Описание |
|--------|--------|----------|
| Трава | 🌿 | Ресурс для травоядных |
| Травоядное | 🐄 | Ищет траву, восстанавливает HP после еды |
| Хищник | 👹 | Охотится на травоядных, атакует и снижает их HP |
| Камень | 🪨 | Статичное препятствие |
| Дерево | 🌳 | Статичное препятствие |

## Архитектура

Поведение описывается через **Action** — каждое действие отдельный класс:
- `initActions` — выполняются один раз перед стартом (расстановка объектов)
- `turnActions` — выполняются каждый ход (движение существ, пополнение травы и травоядных)

```
Entity
├── Grass / Rock / Tree
└── Creature
    ├── Herbivore
    └── Predator
```

## Управление

| Клавиша | Действие |
|---------|----------|
| `F` | Пауза / Продолжить |
